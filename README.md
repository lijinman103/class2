## 定义

信息素养的本质是全球信息化需要人们具备的一种基本能力。开展信息素养课程是为了培养学生的能力：对信息的获取、加工、管理、表达与交流；对信息及信息活动的过程、方法、结果及进行评价；发表观点、交流思想、开展合作与解决学习和生活中实际问题······
